# 利支付

## 项目目标

打造千万级高可用支付平台，实现开箱即可用，简化个人/企业支付研发成本。

开源版本不代表最终品质，文档或定制化需求请加QQ群联系开发者获取

## 项目功能

1. 微信支付快速接入解决方案
2. 支付宝快速接入解决方案
3. 银行卡支付快速接入解决方案
4. SDK支持
5. 高并发能力支持
6. 高可用能力支持

## 模块划分

| 模块                   | 功能描述  |
|----------------------|-------|
| fastpay-simple-admin | 管理系统  |
| fastpay-simple-api   | 交易api |
| fastpay-simple-app   | 交易核心  |
